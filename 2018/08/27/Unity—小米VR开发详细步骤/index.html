Unity—小米VR开发步骤

发表于 2018-08-27 | 分类于 Unity

简介

小米的VR性能很不错，开发设置却非常反人类，环环相扣，少任何一步程序就运行不起来，只能绝望的带着头盔看着眼前的黑屏思考人生。

基本设置

  1. 一台全新的，没有绑定任何设备的小米vr,开机。屏幕显示绑定手机
  2. 手机扫描小米VR说明手册上的二维码，下载小米vr app。
  3. 打开小米vr app,点击添加设备，同时按下小米VR的音量- 键。
  4. 提示连接成功，长按手柄上的○键连接手柄。
  5. 在小米VR开发者平台申请开发者。
  6. 选中手机上小米VR App的设置，关于，出现小米VR的logo,连续点击5次开启小米VR开发者模式
  7. 将usb线与电脑向连接，用adb devices测试是否成功连接。
  8. 若出现一传序列号，则连接成功。若未成功，按照上面步骤检查，或者试着换一条数据线
  9. 将这串序列号保存下来，之后要用

开发准备

  1. 在小米开发者官网下载Mi VR SDK for Unity
  2. 在小米VR开发者管理平台 注册VR的应用号和id.
  3. 在应用的设备签名的选项下，将之前出现序列号输入。下载应用签名文件。点击下载设备签名，
  4. 将签名文件放在 Project/Assets/Plugins/Android/assets/MiVR_sig/ 目录下。目录下没有这个文件夹，创建一个放进去。

开发简述

  1. 具体的文档：【开发文档】
  2. 参考Mi VR SDK for Unity自带的场景，360ViewController.发现小米把我们需要的东西基本封装成prefab了
  3. Prefab/MiCamera是VR摄像机，直接拖出来就可以使用
  4. ControllerLoader是控制器
  5. EventSystem上挂着PointerInputModule
  6. 按照安卓开发流程开发即可。
